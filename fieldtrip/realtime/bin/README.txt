This directory contains compiled executables for working with the FieldTrip
real-time buffer. The files are organized per platform, with names reflecting
the names of MATLAB mex files:

- win32: Contains 32-bit binaries for Microsoft Windows. *Use these files as
  well on 64-bit variants!*
- glnxa64: Linux 64-bit binaries. Mainly tested on CentOS 5.2).
- glnx86: Linux 32-bit (contains a few legacy applications).
- maci64: 64-bit executables for Apple's OS X.
- maci: 32-bit executables for Apple's OS X.
- raspberrypi: Executables for the educational system-on-a-chip 'Raspberry Pi'.
